### bootstrap.yml
Local config
```yaml
info:
  component: Spring Cloud Zuul Server

spring:
  application:
    name: artemis-service-zuul
  cloud:
    config:
      uri: http://localhost:8900

server:
  port: 8902
```

### artemis-service-zuul.yml
Configuration from Config Server
```yaml
zuul:
  routes:
    service-name:
      path: /path_to_service/**
      url: http://service_host_and_port   
```
